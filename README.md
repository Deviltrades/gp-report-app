# GP Report App

A simple, dignified way to log chronic pain and symptoms. Create clear reports for your next appointment.

## Features

- **Symptom Logger**: Log onset, severity, duration, and triggers.
- **Visual Charts**: Track pain severity trends over time.
- **GP Report**: Generate a clinical-style summary for your doctor.
- **Secure**: Data lives in your local browser session (for this prototype).

## Tech Stack

- React
- Vite
- Tailwind CSS
- Shadcn UI
- Recharts
